Crypto Tycoon Bot

Crypto Tycoon Bot — это Telegram-бот в стиле тайкона, где игроки могут покупать видеокарты, улучшать их и зарабатывать криптовалюту. Бот написан на Python с использованием Pyrogram.

Функционал

Главное меню для навигации по разделам игры.

Создание и удаление аккаунта.

Покупка видеокарт — каждая приносит криптовалюту.

Система апгрейдов видеокарт: доход увеличивается при улучшении.

Автоматическое сохранение прогресса в базе данных SQLite.

Команды

/start — начало игры / главное меню

/help — показать справку

/shop — магазин видеокарт и апгрейдов

/profile — Профиль

Установка и запуск

Клонируйте репозиторий и перейдите в папку проекта:

git clone <https://github.com/Yakir12343/Crypto_taikon_bot.git>
cd <папка_проекта>


Создайте виртуальное окружение и активируйте его:

Windows:

python -m venv venv
.\venv\Scripts\activate


Linux/Mac:

python -m venv venv
source venv/bin/activate


Установите зависимости:

pip install -r requirements.txt


Запустите бота:

python main.py

База данных

Используется SQLite для хранения:

Баланса криптовалюты

Количества видеокарт

Уровней апгрейдов

Количества домов, машин, питомцев, развлечений

Настройки

Перед запуском убедитесь, что в файле config.py указаны:

API_ID и API_HASH от Telegram

BOT_TOKEN вашего бота

Особенности

Интерактивные кнопки для управления

Простой и удобный интерфейс

Полностью автономная работа без сторонних серверов
